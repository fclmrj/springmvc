# springmvc - Spring MVC API

Project that explore Spring MVC features.

## Requirements

1. Java - 1.8.x

2. Maven - 3.3.9

## Steps to Setup

**1. Clone the application**

```bash
git clone https://github.com/fclmrj/springmvc.git
```
**2. Build and run the app using maven**

```bash
mvn package
```
